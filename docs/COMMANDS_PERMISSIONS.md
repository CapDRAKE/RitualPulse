# Commandes & permissions

## Commandes joueur
- `/ritual` → ouvre la GUI
- `/ritual claim` → claim direct

## Commandes admin
- `/ritual reload` → reload config/lang + relance le doctor de config
- `/ritual admin doctor` → diagnostic config (warnings lisibles)
- `/ritual admin inspect <player>`
- `/ritual admin reset <player>`
- `/ritual admin givefreeze <player> <amount>`
- `/ritual admin setstreak <player> <value>`

## Permissions
- `ritualpulse.use` (default: true)
- `ritualpulse.admin` (default: op)
- `ritualpulse.bypasscooldown` (default: op)

## Retour admin utiles
- Le plugin log un résumé des warnings config au startup/reload.
- `/ritual admin doctor` affiche les warnings directement en jeu + détails en console.
